Home

Nanoboot is collection of free software mostly written in Java programming language.
  • This software is mostly licenced under GPL or LGPL licences we wish sharing this software, gratis price is 0, your freedom is respected
  • Nanoboot Project was publicly announced on 2nd December 2022 but software development started already in January 2016.
  • You can read the Glossary with the list of terms used across the Nanoboot project.
What we do:
  • Games
  • Web applications Nanodata, Octagon, Color Shapes Archive
  • Desktop applications
  • Tools backups, FTP
  • Tools for programmers DB schema versioning, documentation generator
  • Archiving web pages related to computer game Color linez - colorshapesarchive.nanoboot.org and archive.nanoboot.org
  • Encyclopedia - nanodata.nanoboot.org
