Contribute to Nanoboot

Let's create together the next generation of Nanoboot software.

Everyone Can Contribute

All ideas, improvements, discussions, etc. are welcomed.

What if I do not yet know something

If you would like to start programming, improving documentation, improving this website, etc..
  • But you did not yet learn it.
Then we can help you.
  • Please, contact us in this case at nanoboot at nanoboot.org and we will send you links to websites, where you will be able to gain the knowledge.

How to contribute:

Development

Join us as developer and send code patches/improvements.
  • How to: Create forks of Nanoboot repositories and then ask merge requests. Or send us Git patches.

More info on fork/merge way of collaboration:

  • on a Atlassian webpage
  • on a Github webpage

Documentation

Improve the documentation.

Our documentation is written mostly in Asciidoc.
  • We use the Dog tool to generate static website. Dog is one of Java applications developed by Nanoboot.

Website

Improve this website. Fork its repository: https://code.nanoboot.org/nanoboot/nanoboot-web

Forum

Discuss ideas, topics, improvements ... on our forum: https://forum.nanoboot.org
